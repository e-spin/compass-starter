# compass Starter für SCSS und SASS (Windows)

Mit dem compass Starter wird der watch-Befehl per Batch-Datei gestartet. Man muss auf der Konsole keine Pfade o.ä. eintragen - Doppelklick auf die Datei genügt.

## Installation
Die Batch-Datei in das Basisverzeichnis der CSS-Entwicklung kopieren. Es wird von folgender Struktur ausgegangen:

```
/css
/...
/sass
/scss
start_compass-sass.bat
start_compass-scss.bat
```

## Start
Doppelklick auf die entsprechende Datei :-)
